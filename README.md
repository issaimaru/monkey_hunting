# モンキーハンティング

## モンキーハンティング問題とは
モンキーハンティングとは「銃を持ったハンターが木から落ちたサルに弾丸を当てるために、どのように銃を撃てばいいか？」について考えた高校物理の典型問題らしいです．<br>

結果からいうと，空気抵抗がない条件下では，弾丸の初速度や狙撃する位置に関係なく最初のサルの位置に向けて撃てば必ず命中します．(証明は知らん．)
